# Push Instructions

The files are committed locally but need to be pushed. Run one of these:

## Option 1: Using GitHub CLI
```bash
cd /tmp/stacy-mockup_repo
gh auth login
git push origin main
```

## Option 2: Using Personal Access Token
```bash
cd /tmp/stacy-mockup_repo
git remote set-url origin https://YOUR_TOKEN@github.com/norrin302/stacey-campaign-mockups.git
git push origin main
```

## Option 3: Manual Upload
Open GitHub Desktop or VS Code and push the changes.
